This project is a laboratory work done by Evgeny Vasiliev, a 1st-year master's student at the Faculty of Computer Science and Cybernetics of Taras Shevchenko Kyiv National University

Website has simple functionality. It is a to-do list where you can store and manage your tasks.

The laboratory task is to learn how to use various automatic testing methods:

  • Unit tests
  • Web test
  • Load tests
  • Ordered test
  • Data-driven test
  • Mock objects testing
